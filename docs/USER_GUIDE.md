# Руководство пользователя

Информационная система медицинской клиники MedClinic включает:
- **мобильное приложение** (Android) — для пациентов;
- **веб-лендинг** — публичный сайт клиники;
- **веб-приложение** — личные кабинеты пациента, врача и регистратора (администратора).

---

## 1. Мобильное приложение (Android)

### 1.1. Обзор интерфейса

Главный экран приложения состоит из нижней навигационной панели с **шестью** основными разделами:

- **Главная** — ближайшие записи, быстрые действия и популярные специализации;
- **Доктора** — каталог врачей с поиском и фильтрацией по специализациям;
- **Помощник** — ИИ-ассистент для предварительной консультации по симптомам;
- **Очередь** — отслеживание позиции в электронной очереди в реальном времени;
- **Карта** — интерактивная карта здания с навигацией до кабинетов;
- **Профиль** — личные данные, история записей и выход из системы.

При отсутствии интернет-соединения на всех экранах, кроме «Карты», отображается уведомление о недоступности сети. Карта работает в офлайн-режиме.

### 1.2. Процесс работы

#### Регистрация в системе

Для начала работы с приложением необходимо пройти регистрацию:

**Шаг 1. Основные данные**

1. На экране входа нажмите «Зарегистрируйтесь»;
2. Заполните обязательные поля:
   - ФИО (Фамилия Имя Отчество — минимум два слова);
   - Email (электронная почта);
   - Номер телефона (формат +7);
   - Пароль (не менее 6 символов) и подтверждение пароля;
3. Установите флажок согласия с условиями использования;
4. Нажмите «Продолжить».

**Шаг 2. Дополнительные данные**

1. Укажите дату рождения (выбор через календарь);
2. Выберите пол (Мужской / Женский);
3. При необходимости введите номер страхового полиса (СНИЛС);
4. Нажмите «Завершить регистрацию».

После успешной регистрации вы автоматически войдёте в систему.

#### Вход в систему

1. Введите Email и пароль;
2. Нажмите «Войти»;
3. При отсутствии аккаунта перейдите по ссылке «Нет аккаунта? Зарегистрируйтесь».

При истечении срока действия сессии или ошибке авторизации приложение автоматически перенаправит на экран входа.

#### Запись на приём к врачу

1. Перейдите в раздел «Доктора» (или используйте быстрое действие «Записаться на приём» на главной);
2. Найдите нужного врача с помощью:
   - строки поиска по имени или специализации (поиск выполняется на сервере);
   - фильтров по специализациям (чипы);
3. Нажмите на карточку врача для просмотра профиля или кнопку «Записаться»;
4. При необходимости выберите услугу из выпадающего списка (опционально);
5. Выберите дату приёма (доступны 14 дней вперёд);
6. Выберите свободное время из доступных слотов;
7. Нажмите «Подтвердить запись»;
8. После успешной записи отобразится экран подтверждения с возможностью:
   - вернуться на главную;
   - перейти к списку «Мои записи».

Запись также доступна из профиля врача (кнопка «Записаться на приём») и из раздела «Помощник» после рекомендации ИИ.

#### Просмотр профиля врача

1. Откройте карточку врача в разделе «Доктора» или с главной;
2. На экране профиля отображаются фото, ФИО, специализация, рейтинг и количество отзывов;
3. Переключайтесь между вкладками:
   - **О враче** — биография и описание;
   - **Отзывы** — отзывы других пациентов;
4. Для записи нажмите «Записаться на приём».

### 1.3. Типовой сценарий использования

1. Пациент открывает приложение и авторизуется;
2. На главном экране видит ближайшие записи или переходит в раздел «Доктора»;
3. Выбирает специализацию «Терапевт» из фильтров;
4. Выбирает врача с высоким рейтингом;
5. Нажимает «Записаться» и выбирает:
   - дату: 25 декабря 2025;
   - время: 10:30;
6. Подтверждает запись;
7. В день приёма открывает раздел «Очередь» для отслеживания позиции;
8. При приближении очереди получает push-уведомление;
9. Открывает раздел «Карта» и строит маршрут от регистратуры до кабинета врача (или нажимает «Навигация до кабинета» в деталях записи);
10. После приёма оставляет отзыв о враче.

### 1.4. Работа с электронной очередью

1. Перейдите в раздел «Очередь»;
2. Отобразится список всех ваших активных очередей;
3. Для каждой очереди показывается:
   - ФИО и специализация врача;
   - ваша позиция в очереди;
   - расчётное время ожидания;
   - статус (ожидание / следующий / ваша очередь);
   - количество человек впереди;
4. При изменении позиции вы получите push-уведомление;
5. При первом входе в раздел приложение запросит разрешение на отправку уведомлений (Android 13+).

При отсутствии активных очередей отображается кнопка «Записаться на приём».

### 1.5. Навигация по зданию

1. Перейдите в раздел «Карта»;
2. Выберите начальную точку (например, «Регистратура»);
3. Выберите конечную точку (кабинет врача);
4. Нажмите «Построить маршрут»;
5. На карте отобразится маршрут с анимированной точкой движения;
6. При переходе с экрана записи маршрут строится автоматически от регистратуры до нужного кабинета.

### 1.6. ИИ-помощник

1. Перейдите в раздел «Помощник»;
2. Опишите симптомы в текстовом поле;
3. Нажмите кнопку отправки сообщения;
4. ИИ-ассистент проанализирует описание и предложит рекомендацию врача и услуги;
5. При необходимости нажмите «Записаться» на карточке рекомендованного врача — откроется экран записи с предзаполненными данными;
6. История чата сохраняется локально;
7. Через меню доступны действия «Очистить чат» и «Обновить справочник».

> **Важно:** ИИ-помощник предоставляет предварительную информацию и не заменяет очный приём врача.

### 1.7. Управление профилем

В разделе «Профиль» доступны:

- просмотр личных данных — ФИО, email;
- статистика — количество записей, оценок, активных очередей;
- **Редактировать профиль** — изменение email, телефона, ФИО, даты рождения, пола, СНИЛС;
- **История посещений** — просмотр всех прошедших и предстоящих приёмов;
- **Выйти** — завершение сеанса.

### 1.8. Просмотр и отмена записей

1. Перейдите в «Профиль» → «История посещений» (или «Смотреть все» на главной);
2. Используйте вкладки: **Все**, **Предстоящие**, **Завершены**, **Отмененные**;
3. Выберите нужную запись из списка;
4. На экране деталей отображается:
   - информация о враче (ФИО, специализация);
   - дата и время приёма;
   - номер кабинета;
   - услуга и стоимость;
   - статус записи;
   - диагноз (после приёма);
   - причина отмены (для отменённых записей);
5. Для предстоящих записей доступны:
   - «Навигация до кабинета» — переход на карту с построенным маршрутом;
   - «Отменить запись» — выбор причины отмены и подтверждение;
6. Для завершённых записей доступна кнопка «Оценить и оставить отзыв» (рейтинг 1–5 и комментарий);
7. Для завершённых и отменённых записей доступна кнопка «Повторить» — повторная запись к тому же врачу.

## 2. Веб-лендинг (публичный сайт клиники)

### 2.1. Обзор интерфейса

Лендинг — информационный сайт клиники без авторизации. Основные разделы доступны через верхнее меню:

- **О клинике** (`/about`) — история, миссия, ценности, контакты;
- **Услуги** (`/services`) — каталог медицинских услуг с ценами;
- **Врачи** (`/doctors`) — список специалистов клиники;
- **Запись на приём** — кнопка в шапке сайта, ведущая на страницу онлайн-записи (веб-приложение).

В подвале сайта размещены ссылки на юридические документы: политика конфиденциальности и условия использования.

### 2.2. Главная страница

Главная страница (`/`) состоит из секций:

1. **Hero-блок** — заголовок, описание клиники, кнопка «Запись на приём», ссылка «Связаться с нами»;
2. **Услуги** — превью популярных услуг с ценами;
3. **Команда** — карточки врачей с кнопкой «Записаться» (карусель);
4. **Преимущества и отзывы** — ключевые преимущества клиники и отзывы пациентов;
5. **Мобильное приложение** — информация о мобильном клиенте MedClinic.

### 2.3. Процесс работы

#### Просмотр услуг

1. Перейдите в раздел «Услуги»;
2. Используйте фильтр по специализациям (чипы) для отбора нужных услуг;
3. Для каждой услуги отображаются название, описание, цена и длительность;
4. Нажмите «Записаться» напротив нужной услуги для перехода к онлайн-записи.

#### Поиск врача

1. Перейдите в раздел «Врачи»;
2. Введите имя или специализацию в строку поиска;
3. При необходимости измените сортировку (по имени или по стажу, по возрастанию/убыванию);
4. На карточке врача отображаются фото, ФИО, специализация, стаж и краткая биография;
5. Нажмите «Записаться» для перехода к онлайн-записи.

#### Контактная информация

1. Перейдите в раздел «О клинике»;
2. Прокрутите до секции «Контакты»;
3. Доступны адрес клиники, телефон (кликабельная ссылка для звонка) и email (кликабельная ссылка для отправки письма);
4. Кнопка «Записаться на приём» ведёт на страницу онлайн-записи;
5. Кнопка «Наши врачи» ведёт в каталог специалистов.

### 2.4. Типовой сценарий использования

1. Пользователь заходит на главную страницу сайта клиники;
2. Знакомится с услугами и отзывами на главной;
3. Переходит в раздел «Врачи» и находит терапевта через поиск;
4. Нажимает «Записаться» на карточке врача;
5. Переходит в веб-приложение для завершения записи (требуется регистрация или вход);
6. При необходимости звонит в клинику по телефону из раздела «О клинике».

## 3. Веб-приложение — Пациент

### 3.1. Обзор интерфейса

После входа пациент попадает в личный кабинет. Верхнее меню содержит разделы:

- **Главная** — сводная информация и ближайший приём;
- **Мои приёмы** — список предстоящих и прошедших записей;
- **Запись на приём** — форма онлайн-записи;
- **Профиль** — личные и медицинские данные.

### 3.2. Процесс работы

#### Регистрация

1. На странице входа нажмите «Регистрация»;
2. Заполните поля:
   - Email;
   - ФИО;
   - Телефон (формат +7);
   - Пароль (не менее 8 символов, заглавная и строчная буква, цифра);
   - Подтверждение пароля;
3. Нажмите «Зарегистрироваться»;
4. После успешной регистрации выполните вход с указанными данными.

#### Вход

1. Введите Email и пароль;
2. Нажмите «Войти»;
3. Система автоматически перенаправит в личный кабинет пациента.

#### Запись на приём

Запись выполняется в два шага:

**Шаг 1 — Выбор специалиста и времени**

1. Перейдите в «Запись на приём»;
2. Выберите режим поиска: **по специальности** или **по услуге**;
3. Выберите врача из списка (отображаются фото, рейтинг, специализация);
4. При необходимости просмотрите отзывы о враче;
5. Выберите дату в календаре доступных дат;
6. Выберите свободный временной слот;
7. При необходимости укажите услугу;
8. Нажмите «Далее».

**Шаг 2 — Подтверждение**

1. Проверьте сводку: врач, дата, время, услуга, стоимость;
2. Укажите причину обращения (необязательно);
3. Выберите способ напоминания: email или «не отправлять»;
4. Нажмите «Подтвердить запись».

#### Просмотр и отмена записей

1. Перейдите в «Мои приёмы»;
2. Используйте вкладки «Предстоящие» и «История»;
3. Для каждой записи отображаются врач, специализация, кабинет, услуга и цена;
4. Для предстоящих записей нажмите «Отменить» для отмены приёма;
5. Для завершённых записей нажмите на запись для просмотра деталей (диагноз, статус).

#### Управление профилем

1. Перейдите в «Профиль»;
2. На вкладке **Личные данные** измените ФИО, дату рождения, пол, телефон;
3. На вкладке **Медицинская информация** укажите номер полиса ОМС;
4. Нажмите «Сохранить».

### 3.3. Типовой сценарий использования

1. Пациент переходит на сайт клиники и нажимает «Запись на приём»;
2. Регистрируется или входит в систему;
3. На главной странице видит ближайший запланированный приём;
4. Переходит в «Запись на приём», выбирает специальность «Терапевт»;
5. Выбирает врача, дату и время;
6. Подтверждает запись с указанием причины обращения;
7. В день приёма проверяет детали в разделе «Мои приёмы»;
8. При необходимости отменяет запись через тот же раздел.

## 4. Веб-приложение — Врач

### 4.1. Обзор интерфейса

Личный кабинет врача содержит раздел:

- **Мои приёмы** — расписание приёмов на выбранную дату и живая очередь пациентов.

Учётная запись врача создаётся администратором клиники; самостоятельная регистрация недоступна.

### 4.2. Процесс работы

#### Вход

1. Введите Email и пароль, выданные администратором;
2. Нажмите «Войти»;
3. Система перенаправит на страницу «Мои приёмы».

#### Просмотр расписания

1. На странице «Мои приёмы» выберите дату через календарь;
2. В таблице отображаются все приёмы на выбранный день:
   - время;
   - ФИО пациента;
   - статус;
   - диагноз (если приём завершён);
3. Нажмите на строку приёма для перехода к детальной карточке.

#### Работа с карточкой приёма

1. Откройте приём, нажав на строку в таблице;
2. Просмотрите данные: пациент, услуга, кабинет, время;
3. При необходимости измените:
   - статус (запланирован, подтверждён, в процессе, завершён, отменён, не явился);
   - диагноз по МКБ-10 (поиск по справочнику);
   - жалобы, анамнез, назначения и план лечения;
   - причину отмены;
4. Нажмите «Сохранить».

#### Завершение приёма

1. На странице «Мои приёмы» найдите текущий приём;
2. Нажмите «Завершить»;
3. В диалоговом окне выберите диагноз из справочника МКБ-10;
4. Подтвердите завершение;
5. Живая очередь автоматически обновится — следующий пациент переместится в начало.

#### Живая очередь

На странице «Мои приёмы» отображается блок **Живая очередь**:

- список пациентов в порядке очереди;
- позиция каждого пациента;
- обновление в реальном времени после завершения приёмов.

### 4.3. Типовой сценарий использования

1. Врач входит в систему утром перед началом смены;
2. Выбирает текущую дату и просматривает список запланированных приёмов;
3. Проверяет живую очередь — видит порядок пациентов;
4. Принимает первого пациента, открывает карточку приёма;
5. Заполняет жалобы, анамнез, назначения;
6. Нажимает «Завершить», указывает диагноз МКБ-10;
7. Очередь автоматически сдвигается к следующему пациенту;
8. Повторяет процесс для каждого приёма в течение дня.

## 5. Веб-приложение — Регистратор (администратор)

> В системе роль регистратора реализована как **администратор** (`admin`). Доступ ко всем функциям управления клиникой предоставляется через административную панель.

### 5.1. Обзор интерфейса

После входа администратор попадает в панель управления. Верхнее меню содержит разделы:

- **Главная** — сводная статистика и быстрые действия;
- **Врачи** — управление карточками врачей и расписанием;
- **Пациенты** — управление карточками пациентов;
- **Категории** — справочник специализаций;
- **Услуги** — справочник медицинских услуг;
- **Приёмы** — мониторинг и управление записями;
- **Статистика** — отчёты и экспорт данных.

### 5.2. Процесс работы

#### Вход

1. Введите Email и пароль администратора;
2. Нажмите «Войти»;
3. Система перенаправит на панель управления.

#### Панель управления (Dashboard)

На главной странице отображаются:

- количество врачей, пациентов и приёмов на сегодня;
- список сегодняшних приёмов;
- быстрые действия: добавить пациента, добавить врача, перейти к справочникам.

#### Управление пациентами

1. Перейдите в «Пациенты»;
2. Используйте поиск по ФИО, полису или телефону;
3. Доступные действия для каждого пациента:
   - **Изменить** — редактирование данных;
   - **Записать** — запись на приём от имени пациента;
   - **Удалить** — удаление карточки.

**Создание нового пациента:**

1. Нажмите «Добавить пациента»;
2. Заполните данные: ФИО, дата рождения, пол, телефон, email, полис;
3. На втором шаге система автоматически сгенерирует логин и пароль;
4. Сохраните учётные данные и передайте их пациенту.

**Запись пациента на приём:**

1. Нажмите «Записать» напротив нужного пациента;
2. В диалоговом окне выберите врача, дату, время и услугу (аналогично форме пациента);
3. Подтвердите запись.

#### Управление врачами

1. Перейдите в «Врачи»;
2. Используйте поиск по ФИО, специальности, email или телефону;
3. Доступные действия:
   - **Изменить** — редактирование карточки;
   - **Расписание** — настройка рабочего расписания;
   - **Отзывы** — просмотр отзывов пациентов;
   - **Удалить** — удаление карточки.

**Создание нового врача:**

1. Нажмите «Добавить врача»;
2. Заполните: ФИО, телефон, email, стаж, биографию;
3. Выберите специализации;
4. Загрузите фото;
5. Сохраните карточку.

**Настройка расписания:**

1. Нажмите «Расписание» напротив врача;
2. Укажите рабочие дни и часы приёма;
3. Сохраните изменения.

#### Управление приёмами

1. Перейдите в «Приёмы»;
2. Используйте фильтры: дата, врач, статус;
3. В таблице отображаются все записи с пагинацией;
4. Нажмите на запись для просмотра деталей в диалоговом окне.

#### Справочники

**Категории (специализации):**

1. Перейдите в «Категории»;
2. Создайте, измените или удалите специализацию (код, название, описание);
3. Используйте поиск для быстрого нахождения.

**Услуги:**

1. Перейдите в «Услуги»;
2. Создайте, измените или удалите услугу (код, название, цена, описание);
3. Привяжите услугу к специализациям;
4. Используйте поиск для фильтрации.

#### Статистика и отчёты

1. Перейдите в «Статистика»;
2. Выберите период: один день или диапазон дат;
3. При необходимости укажите конкретного врача;
4. Выберите тип отчёта:
   - общая статистика;
   - по статусам приёмов;
   - по врачам;
5. Просмотрите графики на странице;
6. Экспортируйте отчёт в **Excel** или **PDF**.

### 5.3. Типовой сценарий использования

1. Регистратор входит в систему в начале рабочего дня;
2. На панели управления проверяет количество приёмов на сегодня;
3. Принимает звонок от нового пациента — создаёт карточку через «Добавить пациента»;
4. Записывает пациента на приём к терапевту через кнопку «Записать»;
5. Передаёт пациенту сгенерированные логин и пароль;
6. При необходимости корректирует расписание врача;
7. В конце дня формирует отчёт за день и экспортирует в PDF;
8. Проверяет статистику по врачам за неделю.
